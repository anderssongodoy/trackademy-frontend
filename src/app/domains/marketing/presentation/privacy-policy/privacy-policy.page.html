Politica de privacidad

Esta politica explica como Trackademy recopila, usa y protege la informacion necesaria para operar tu experiencia academica.

Resumen

Trackademy usa tus datos para mostrar cursos, horarios, evaluaciones, notas, silabos, recordatorios y sincronizacion de calendario.

No vendemos tus datos personales. No usamos informacion de Google Calendar o WhatsApp para publicidad.

1. Responsable del servicio

Trackademy es una aplicacion academica para estudiantes. Para consultas sobre privacidad puedes escribir a anderssongodoygarcia@gmail.com.

2. Informacion que recopilamos

Podemos recopilar las siguientes categorias de informacion:

  • Datos de identidad: nombre, correo, proveedor de acceso y estado de autenticacion.
  • Datos academicos: campus, carrera, periodo, ciclo, cursos seleccionados, horarios, evaluaciones, notas, metas y preferencias.
  • Datos de silabo: versiones disponibles, estado del PDF y contenido academico asociado al curso.
  • Datos de calendario: eventos academicos generados por Trackademy y tokens autorizados por el usuario para sincronizacion.
  • Datos de WhatsApp, si decides vincularlo: numero enmascarado, estado de vinculacion y mensajes necesarios para operar comandos o alertas.

3. Como usamos la informacion

Usamos la informacion para:

  • Autenticar tu acceso a Trackademy.
  • Mostrar tu dashboard, cursos, horario, calendario, notas, tareas y recordatorios.
  • Registrar notas y calcular avances academicos visibles para ti.
  • Sincronizar eventos academicos hacia Google Calendar cuando otorgas permiso.
  • Enviar recordatorios o respuestas por WhatsApp cuando vinculas ese canal.
  • Prevenir errores, abuso, accesos no autorizados y mejorar la estabilidad del servicio.

4. Uso de datos de Google

Si conectas Google, Trackademy puede solicitar permisos para iniciar sesion y para crear o actualizar eventos academicos en Google Calendar. El acceso se limita a lo necesario para sincronizar eventos generados por Trackademy.

Trackademy no usa datos de Google para publicidad, perfilado comercial ni venta a terceros. Los tokens de acceso se almacenan cifrados y se usan solo para mantener la sincronizacion autorizada por el usuario.

5. Uso de WhatsApp

Si vinculas WhatsApp, Trackademy puede usar ese canal para responder comandos, enviar recordatorios academicos y mostrar el estado de vinculacion. No vendemos ni compartimos tu numero para publicidad.

6. Con quien compartimos informacion

No vendemos datos personales. Podemos procesar informacion mediante proveedores necesarios para operar el servicio:

  • Google, cuando usas inicio de sesion o sincronizacion de Google Calendar.
  • Meta/WhatsApp, cuando vinculas WhatsApp para mensajes o recordatorios.
  • Servicios de infraestructura, hosting, base de datos y monitoreo necesarios para ejecutar Trackademy.

7. Retencion y eliminacion

Conservamos tus datos mientras tu cuenta este activa o mientras sean necesarios para operar el periodo academico. Puedes solicitar eliminacion de datos escribiendo al correo de contacto. Si desconectas Google Calendar o WhatsApp, dejaremos de usar esos permisos para nuevas operaciones.

8. Seguridad

Aplicamos controles razonables para proteger la informacion, incluyendo autenticacion, autorizacion y cifrado de tokens externos. Ningun sistema es completamente infalible, pero limitamos el acceso a lo necesario para operar Trackademy.

9. Cambios a esta politica

Podemos actualizar esta politica cuando cambie el producto, los proveedores o las obligaciones aplicables. La version vigente siempre estara disponible en esta pagina.
